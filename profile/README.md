# Fortran index: a community initiative for improving knowledge exchange related to the Fortran programming language

This organisation includes repositories supporting the Fortran index hackathon
series:

* [Fortran index repo](https://github.com/fortran-index/fortran-index): Core
  repository containing information related to how things work in the hackathon
  series and documentation source for the
  [https://fortran-index.github.io](https://fortran-index.github.io) webpage.
* [Fortran index GitHub Pages root](https://github.com/fortran-index/fortran-index.github.io):
  Redirect from
  [https://fortran-index.github.io](https://fortran-index.github.io) to the docs
  found at
  [https://fortran-index.github.io/fortran-index](https://fortran-index.github.io/fortran-index).
* [Fortran-lang webpage fork](https://github.com/fortran-index/fortran-lang-webpage):
  Fork of the GitHub repository for the
  [https://fortran-lang.org](https://fortran-lang.org) community webpage.
